# Thinking in React

Clojurescript port of JS in https://www.udemy.com/course/react-clarified-thinking-in-react/

> Built with shadow-cljs and Reagent

## Run

``` shell
npm install

npm run watch
```

## Clean

``` shell
npm run clean
```

## Release

``` shell
npm run release
```
